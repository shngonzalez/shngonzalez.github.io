Prototyping AZLyrics

// Website redesign, course project, HTML & CSS

November 2018

Summary

AZLyrics is a popular website where users can find song lyrics. Its outdated design
and unintuitive features call for restructuring from the ground up.

For this website redesign, I created a prototype that converts AZLyrics to a modern
website highlighting its searching function. Out of approx. 100 students, my team
ranked in the top 5 for the class project competition.

My Role

  • Prototyping
  • UI/UX Design
  • Wireframing

    • Overview

    This is the first website prototype I created as part of the Web Design DeCal, which is
    the class that inspired me to pursue web development. I worked in a three-person
    team to redesign 4 pages of AZLyrics: Home, Search, Artist and Lyrics views. I was in
    charge of the navigation bar, the home page, and the lyrics view.

    Finding a Theme

    We decided to aim for a dark Spotify-like theme, going with a grey background with a
    hint of bright colors to highlight content. Choosing a blue-green gradient as an accent
    color amplified the music atmosphere we wanted.

    Polishing Wireframes

    After deciding the theme, I created a mood board consisting of design elements that
    matched the theme. Based on those, I designed wireframes in Adobe XD and coded it
    in HTML. I talked to the instructor for feedback, and fixed the wireframes accordingly.

    Keeping "A to Z"

    The first problem I encountered was deciding whether to keep the A to Z list or not.
    In the original design, there is a list of clickable buttons from A to Z in the navigation
    bar listing artist names that start with that letter. It felt a little unintuitive to use.
    Afterall, the website’s main function is for searching lyrics that the user is already
    familiar with. However, my teammates and I agreed to keep the A to Z because the
    website is named “AZ” Lyrics. For trademark reasons, this feature remains on the
    navigation bar.

    However, I still wanted to improve the A to Z function. To do so, it had to be more
    subtle yet interactive. The font color is now a lighter grey, with the hovering effect
    changing its color to a bright blue-green shade. Having this contrast increased visual
    feedback to the user. A little change can increase interactivity.

    Uncluttering the Homepage

    AZLyrics’ main function is searching for lyrics, so the search bar should be the thing
    popping out of the homepage. Instead, AZLyrics’ homepage is cluttered with the
    “What’s Hot” feature, and the search bar barely catches the user’s eyes.

    I decided to remove the “What’s Hot” section to highlight the search bar. For users
    who’d rather have the feature, they can search the album titles seen in the
    background. This way, the clutter is cleared, and the user is pushed to focus on the
    site’s main function: searching.

    Readability and Function

    The lyrics view was simple and straightforward. I focused on readability and
    interactivity. I increased the contrast between the text and the background and
    added more space in between the lines.

    A common problem when viewing lyrics is that people tend to lose track of where
    they are. To solve this, I added a hovering effect to highlight lines as people read/sing
    along.

    AZLyrics also lacked useful features frequently found in lyric websites: printing, liking,
    and sharing the lyrics. I added these features in a little block on the side, which
    follows along the page as the user scrolls through.

    Obstacles & Learnings

    Working with other designers demanded that we all know which color, spacing, and
    size guidelines to follow so that the whole product looks consistent. In the end, we
    all used different numbers and the product had varying spacings and sizes. It was
    obvious that each page was designed separately.

    From this experience, I learned the importance of setting up common design
    guidelines before starting a team project to make sure that everyone in the group is
    on the same page.
